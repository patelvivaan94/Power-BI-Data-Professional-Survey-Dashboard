**Workforce Survey Analytics Dashboard (Power BI)**

**Overview**

This project transforms raw workforce survey data into clear, decision-ready insights using Power BI. The goal was to improve data quality, enable reliable reporting, and deliver an interactive dashboard that highlights compensation trends, job satisfaction, and technical skill distributions.

The final output supports consistent analysis across 630+ survey responses and is designed for long-term reuse and reporting.

**What This Project Demonstrates**
- Strong data cleaning and transformation skills
- Practical business analytics and reporting judgment
- Ability to turn messy data into executive-ready insights
- Clear documentation and data governance practices

**Data Preparation & Transformation**
- Cleaned, transformed, and standardized raw CSV survey data using Power Query
- Resolved inconsistent job titles, salary formats, and survey responses
- Applied documented data quality rules to ensure consistent interpretation across all visuals

**Analysis & Modeling**
- Converted text-based salary ranges into calculated numeric fields
- Enabled accurate aggregation and comparison across roles, experience levels, and demographics
- Ensured analytical consistency across all measures and visuals

**Dashboard & Insights**
- Designed a polished, interactive Power BI dashboard focused on:
  - Compensation trends
  - Job satisfaction patterns
  - Technical skill distribution
- Prioritized readability, usability, and stakeholder clarity
- Delivered business-ready insights suitable for decision-makers

**Tools & Technologies**
- Power BI
- Power Query
- CSV data sources

**Key Outcome**

A reliable, reusable analytics solution that transforms raw survey data into structured insights, enabling accurate reporting and informed workforce decision-making.
